</div>

Telegram-бот для отслеживания криптовалют с уведомлениями и управлением портфелем.

## 📋 Функционал

- 💰 **Актуальные цены** - получение текущей цены любой криптовалюты
- 🔔 **Уведомления** - установка ценовых порогов (выше/ниже) с автоматическими оповещениями
- 📊 **Криптопортфель** - добавление/удаление монет и отслеживание общей стоимости
- 📱 **Две клавиатуры** - обычная и инлайн для удобного управления
- 🖼️ **Приветственное изображение** - стильное приветствие с логотипом

## 🛠️ Технологии

- **Python 3.11+**
- **aiogram 3.x** - асинхронный фреймворк для Telegram Bot API
- **CoinGecko API** - получение данных о криптовалютах
- **SQLite** - хранение настроек пользователей
- **APScheduler** - фоновые задачи для проверки цен
